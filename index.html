핵심 요구사항 분석

  • 응용분야 중심 IA — 단순 제품 카탈로그가 아닌, 고객의 연구 목적과 응용 분야 기반으로 제품·솔루션을 연결하는 구조
  • 제품 ↔ 응용분야 N:N 자동 연결 — 한 제품이 여러 응용분야에 노출되고, 응용분야 페이지에서 관련 제품/자료가 자동 표시
  • 응용분야 스토리텔링형 UI — Challenge → Solution → Reference → Product 흐름으로 구성
  • 통합 검색 — 제품명/게시글 제목/본문/첨부파일명 검색 + 브랜드·응용분야 기반 필터링
  • 자료실 — Document / News / Webinar / Video 4종 + PDF·이미지·Video 다운로드 + 웨비나 다시보기
  • 마케팅 기능 — SEO/GEO 최적화, Open Graph·메타태그 관리, URL 커스텀, 팝업/프로모션 노출
  • Salesforce MCAE CRM 연동 — 문의·데모 신청·교육 신청 폼 데이터 자동 전송
  • GA4 방문자 분석 — 유입 경로/페이지별 전환 대시보드
  • 관리자 CMS 고도화 — 제품/응용분야/자료실/팝업/SEO 메타/문의 데이터 통합 관리
  • 관리자 교육 + 운영 매뉴얼 — 내부 마케팅팀이 직접 콘텐츠 운영 가능하도록 지원
  • 최신 웹 표준 + SSL(HTTPS) — 접근성, 페이지 속도(Lighthouse 90+), 보안 기준 준수

제안 포인트

1. 참고 사이트 분석 기반 IA 설계 — 클라이언트가 제시한 Cytiva Korea, LG에너지솔루션 Battery Inside, 자연과학 자료실 3개 사이트를 직접 분석하여, 응용분야(mAb·바이러스 벡터·mRNA 등 연구 목적별) 카테고리 + 제품 기술 기반 분류 + 자료실 5종 분류(News/Document/Webinar/Video/Others) + 브랜드·응용분야 필터 + 3종 뷰 모드(gallery·magazine·list) 패턴을 반영합니다.

2. Astra Pro 기반 경량 테마 + 자체 Gutenberg Block 병행 — 일반 콘텐츠 페이지는 Elementor Pro로 운영팀이 직접 편집하고, 메인 히어로/응용분야 스토리텔링/제품 비교 등 핵심 페이지는 자체 Custom Gutenberg Block으로 구현하여 페이지 로딩 1.5초 이내, Lighthouse 점수 90+를 동시에 달성합니다.

3. 제품 ↔ 응용분야 N:N 자동 연결 — ACF Pro Bidirectional Relationship 필드를 활용하여 한 번의 등록으로 양방향 관계가 자동 구성됩니다. 제품 등록 시 응용분야를 체크하면 응용분야 페이지에 해당 제품이 자동 노출되고, 자료실 콘텐츠도 동일 메커니즘으로 연결됩니다.

4. Salesforce MCAE 안전한 연동 — Web-to-Lead 표준 엔드포인트 + reCAPTCHA v3 스팸 방지 + 서버사이드 토큰 검증으로 보안과 안정성 확보. 폼 제출 실패 시 워드프레스 DB에 백업 저장하여 데이터 유실 0% 보장.

5. SEO + GEO(Generative Engine Optimization) 동시 최적화 — 검색엔진(Google/Naver) + 생성형 AI 엔진(ChatGPT/Perplexity) 동시 노출을 위해 Schema.org 구조화 데이터(Product, Organization, FAQ, BreadcrumbList) 적용 + JSON-LD + llms.txt + 의미 단위 콘텐츠 청크 구조 적용.

6. 관리자 운영 자율성 확보 — 향후 내부 마케팅팀이 직접 콘텐츠를 운영할 수 있도록 화상 교육(2회) + 영상 녹화본 + 단계별 스크린샷 매뉴얼 제공. Elementor 기반 콘텐츠 편집은 비개발자도 즉시 활용 가능.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

배경 — 생명과학 연구 및 분석 장비를 중심으로 한 공식 홈페이지가 단순 기업 소개 형태에 머물러 있어, 고객의 연구 목적과 응용 분야 기반 탐색이 어렵고 잠재 고객 데이터 수집 채널이 부재합니다.

목적

  • 응용분야(Application) 중심 IA로 고객의 연구 여정에 맞춘 정보 구조 제공
  • 글로벌 기술 기업 수준의 전문적이고 신뢰감 있는 브랜드 이미지 구축
  • 잠재 고객(Lead) 확보 및 세일즈 전환율 향상
  • 모바일 최적화 + CMS 고도화 + Salesforce MCAE 연동을 통한 데이터 기반 운영 체계 구축
