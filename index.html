제안 포인트

1. 응용분야 중심 IA로 잠재 고객 여정 재설계

Cytiva, LG에너지솔루션 Inside 등 글로벌 B2B 기술 기업 사이트 분석 결과, 단순 제품 카탈로그가 아닌 고객의 연구 목적·응용분야 → 솔루션 → 제품으로 이어지는 워크플로우 중심 IA가 신뢰도와 전환율을 동시에 높이는 핵심임을 확인하였습니다. 본 프로젝트도 응용분야 페이지를 Challenge → Solution → Reference → Product 흐름으로 구성하여 잠재 고객(Researcher / PI / 구매 담당자)이 자연스럽게 제품 페이지로 진입하도록 설계합니다.

2. 제품 ↔ 응용분야 N:N 자동 연결 — 콘텐츠 운영 효율 극대화

ACF Pro의 양방향 관계(Bidirectional Relationship) 기능과 커스텀 포스트 타입(CPT)을 활용하여 제품과 응용분야를 한 번만 매핑하면 양쪽 페이지에서 자동 노출되도록 구축합니다. 자료실(Document · News · Webinar · Video)도 동일한 분류 체계(Taxonomy)에 연결하여, 제품 → 자료실 → 응용분야 상호 연결 구조를 만듭니다. 콘텐츠가 늘어날수록 운영 효율과 SEO 가치가 모두 증가하는 구조입니다.

3. 통합 검색 — 본문·첨부파일·필터까지 한 번에

자연과학 자료실(reference-room) 패턴을 참고하여 Relevanssi 또는 ElasticPress 기반 통합 검색을 구축합니다. 제품명 · 게시글 제목 · 본문 · 첨부 PDF 파일명까지 검색 대상에 포함하고, 브랜드 / 응용분야 필터로 결과를 좁힐 수 있도록 합니다. 인기 키워드 추천, Gallery / Magazine / List 디스플레이 모드도 함께 제공합니다.

4. Salesforce MCAE 연동 — 마케팅 → 영업 자동화 파이프라인

모든 폼(데모 신청 · 세미나 · 웨비나 · 문의)을 Salesforce MCAE(구 Pardot) Form Handler 또는 REST API로 연동합니다. 리드 데이터를 자동으로 캠페인 · 점수 · 할당 룰에 연결하여 영업팀이 즉시 후속 조치할 수 있도록 합니다. UTM · Referrer · GA Client ID도 함께 전송하여 어트리뷰션을 정확히 추적합니다.

5. SEO + GEO(Generative Engine Optimization) 동시 최적화

전통적 SEO(메타 태그, 구조화 데이터, 사이트맵, OG)뿐만 아니라 ChatGPT · Gemini · Perplexity 등 생성형 검색 엔진이 크롤링 · 인용하기 좋은 구조로 콘텐츠를 설계합니다. FAQ 스키마 · HowTo 스키마 · Article 스키마 · Product 스키마 · llms.txt 등을 적용하여 AI 답변에 노출될 가능성을 높입니다.

6. 운영 효율을 고려한 테마 선택 및 커스터마이징

로딩 속도 · 코어 웹 바이탈 · 추후 운영성을 고려해 GeneratePress / Blocksy / Astra Pro 기반 + Bricks Builder 또는 Elementor Pro 조합을 권장합니다. 핵심 템플릿(제품 카테고리 / 제품 상세 / 응용분야 / 자료실)은 ACF Pro + 자체 PHP 템플릿으로 커스터마이징하여 페이지 빌더의 오버헤드 없이 빠르게 동작하면서도, 비개발자가 콘텐츠를 쉽게 편집할 수 있도록 합니다.

7. 유사 B2B 콘텐츠·관리자 시스템 구축 경험 기반

유사 B2B 도메인의 콘텐츠 관리 시스템 / 관리자 포탈 / 외부 CRM API 연동을 다수 구축한 경험을 바탕으로, 클라이언트 예상 일정(90일) 안에 산출물 완성도(1순위)와 일정 준수(3순위)를 동시에 달성할 수 있도록 설계하였습니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

본 프로젝트는 생명과학 연구·분석 장비 중심의 공식 홈페이지를 단순 기업 소개에서 응용분야 기반 솔루션 디스커버리 사이트로 전면 재구성하는 리뉴얼입니다. 모바일 최적화, 직관적인 정보 구조(IA), CMS 고도화, Salesforce MCAE 연동을 통해 데이터 기반 운영 체계를 구축하고, 글로벌 기술 기업 수준의 전문적이고 신뢰감 있는 브랜드 이미지로 잠재 고객(Lead) 확보 및 세일즈 전환율 향상을 목표로 합니다.
